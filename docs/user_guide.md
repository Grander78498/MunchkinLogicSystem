# Руководство пользователя SHLDN

## Введение

SHLDN (Simple Higher-Level Declarative Notation) - это логический интерпретатор, предназначенный для решения задач логического программирования. Он поддерживает исчисление высказываний и исчисление предикатов с унификацией переменных.

## Запуск программы

Для запуска SHLDN выполните команду:
```bash
uv run python repl.py
```

После запуска вы увидите приглашение:
```
SHLDN Логический Интерпретатор
Введите 'help' для получения справки, 'quit' или 'exit' для выхода.

SHLDN>
```

## Основные команды

### help
Выводит справочную информацию о языке и доступных командах:
```
SHLDN> help
```

### load <файл>
Загружает содержимое файла в текущую базу знаний:
```
SHLDN> load examples/simple_example.shldn
```

### get axioms
Выводит все аксиомы, находящиеся в текущей базе знаний:
```
SHLDN> get axioms
```

### remove <номер>
Удаляет аксиому с указанным номером из базы знаний:
```
SHLDN> remove 1
```

### quit или exit
Завершает работу программы:
```
SHLDN> quit
```

## Синтаксис языка

### Высказывания
Высказывания - это идентификаторы, состоящие из букв, цифр и подчеркиваний:
```
игрок_1_уровень_10
монстр_дракон
победа
```

### Переменные
Переменные начинаются с большой буквы:
```
X, Y, Z, Player, Monster
```

### Предикаты
Предикаты - это функции с аргументами:
```
уровень(игрок(1), 2)
сила(игрок(1), 5)
победа(игрок(1), монстр(гоблин))
```

Аргументами предикатов могут быть:
- Идентификаторы
- Переменные
- Числа
- Другие предикаты
- Арифметические выражения

### Логические операции

#### Конъюнкция (И)
Обозначается символами `&` или `*`:
```
игрок_1_уровень_10 & игрок_1_сила_5
игрок_1_уровень_10 * игрок_1_сила_5
```

#### Дизъюнкция (ИЛИ)
Обозначается символами `|` или `+`:
```
игрок_1_вор | игрок_1_волшебник
игрок_1_вор + игрок_1_волшебник
```

#### Отрицание (НЕ)
Обозначается символом `!`:
```
!игрок_1_воин
```

#### Импликация (ЕСЛИ-ТО)
Обозначается символами `->`:
```
игрок_1_уровень_10 -> игрок_1_победитель
```

#### Эквиваленция (ТОГДА И ТОЛЬКО ТОГДА)
Обозначается символами `<->`:
```
игрок_1_уровень_10 <-> игрок_1_сильный
```

### Арифметические операции

#### Сложение
Обозначается символом `+`:
```
Y + Z
2 + 3
```

#### Вычитание
Обозначается символом `-`:
```
Y - Z
5 - 2
```

#### Умножение
Обозначается символом `*`:
```
Y * Z
3 * 4
```

#### Деление
Обозначается символом `/`:
```
Y / Z
10 / 2
```

### Операции сравнения

#### Больше
Обозначается символом `>`:
```
Sx > Sm
7 > 5
```

#### Меньше
Обозначается символом `<`:
```
Sx < Sm
5 < 7
```

#### Больше или равно
Обозначается символами `>=`:
```
Sx >= Sm
7 >= 7
```

#### Меньше или равно
Обозначается символами `<=`:
```
Sx <= Sm
5 <= 7
```

#### Равно
Обозначается символами `==`:
```
Sx == Sm
7 == 7
```

#### Не равно
Обозначается символами `!=`:
```
Sx != Sm
7 != 5
```

### Аксиомы
Аксиомы - это импликации, определяющие логические связи между выражениями:
```
игрок_1_уровень_10 -> игрок_1_победитель
игрок_2_уровень_5 & игрок_2_сила_10 & монстр_сила_14 -> игрок_победил
уровень(игрок(X), Y) & сила(игрок(X), Z) -> боевая_сила(игрок(X), Y + Z)
```

### Теоремы (запросы)
Теоремы - это выражения, истинность которых нужно доказать. Они начинаются с символа `?`:
```
? игрок_1_победитель
? боевая_сила(игрок(1), 7)
? победа(игрок(1), монстр(гоблин))
```

## Примеры использования

### Пример 1: Простая логика
```
SHLDN> игрок_1_уровень_10
Выражение добавлено как аксиома.

SHLDN> игрок_1_уровень_10 -> игрок_1_победитель
Аксиома добавлена.

SHLDN> ? игрок_1_победитель
Доказательство: игрок_1_победитель
(1) !игрок_1_уровень_10 + игрок_1_победитель |
(2) игрок_1_уровень_10                   |
 |--> Пустой дизъюнкт - теорема доказана.
```

### Пример 2: Работа с предикатами
```
SHLDN> уровень(игрок(1), 2)
Выражение добавлено как аксиома.

SHLDN> сила(игрок(1), 5)
Выражение добавлено как аксиома.

SHLDN> уровень(игрок(X), Y) & сила(игрок(X), Z) -> боевая_сила(игрок(X), Y + Z)
Аксиома добавлена.

SHLDN> ? боевая_сила(игрок(1), 7)
Доказательство: боевая_сила(игрок(1), 7)
...
```

### Пример 3: Использование файла
```
SHLDN> load examples/simple_example.shldn
Файл 'examples/simple_example.shldn' успешно загружен

SHLDN> get axioms
Аксиомы в базе знаний:
  1. !игрок_1_уровень_10 + игрок_1_победитель
  2. игрок_победил + !монстр_сила_14 + !игрок_2_уровень_5 + !игрок_2_сила_10
```

## Решение проблем

### Ошибки синтаксиса
Если вы получаете ошибки синтаксиса, проверьте:
- Все скобки закрыты
- Все операторы используют правильные символы
- Все идентификаторы написаны правильно

### Ошибки при доказательстве теорем
Если теорема не доказывается:
- Убедитесь, что все необходимые аксиомы и факты добавлены
- Проверьте, что переменные используются правильно
- Попробуйте разбить сложную теорему на более простые

### Ошибки загрузки файлов
Если файл не загружается:
- Проверьте путь к файлу
- Убедитесь, что файл существует и доступен для чтения
- Проверьте, что файл имеет правильный синтаксис
